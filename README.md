# ggscribe <a href="https://davidhodge931.github.io/ggscribe/"><img src="man/figures/logo.png" align="right" height="139" alt="ggscribe website" /></a>

<!-- badges: start -->

<!-- badges: end -->

The goal of ggscribe is to annotate ‘ggplot2’ visualisation using the
set theme.

This includes functions to annotate components of the set theme, and
functions to annotate that use set theme defaults.

## Installation

You can install the development version of ggscribe from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("davidhodge931/ggscribe")
```
